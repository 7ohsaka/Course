## 0312
tips:  
1. python是动态的，C语言是静态的；  
2. python更灵活，但是结果容易出错；  
3. python对大小写敏感
4. bool型——True和False
5. 空类型 None 不是0没有值
6. // 向下取整
7. 单行字符串''或者''... '';多行字符串 '''...'''
8. 编号从0开始，跟C语言一样
9. python范围是“左闭右开”原则（！分片等操作时应该注意！）
10. 字符串定义之后不能直接改变其中某项，需要使用函数或者切片方法
11. 分片操作高级用法：s[i:j:k] 表示从i开始到j-1项，步长为k
